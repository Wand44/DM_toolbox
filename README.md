# DM Toolbox

A collection of utilities and tools for Dungeon Masters to streamline game preparation and session management.

## Features

- **Character Management**: Tools for creating NPCs
- **Encounter Builder**: Generate random encounters based on party level
- **Treasure Generator**: Create loot tables and magic items on demand
- **Shop Generator**: Create a shop of a seleced type

## Usage 

~~~bash
python3 DM_toolbox_cmd.py
~~~
OR
~~~bash
python3 DM_toolbox.py
~~~
